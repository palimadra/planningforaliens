The Invasion

A weekly(ish) newsletter for web people in the trenches

The web is stuffed with HTML5 evangelists, elite Javascript hackers, CSS3 black belts, mustachioed Node.js hipsters and responsive design gurus. It seems like every day there's something new and enticing to learn about.

The pace of change is overwhelming.

But who doesn't like oohing and aahing over a flashy, HTML5 demo in Chrome Canari? It's fun reading about state of the art web development. At the same time, these kinds of demos and articles are frustrating. What's the point if you never get to put to practice the topics they cover?

Your code has to work in IE8.

Your projects start with piles of legacy code, not a fresh, clean HTML document.

The real web is messy and complicated.

This is why every week or so I write for my newsletter about topics relevant to real web developers, like you and me. We don't own agencies or have big, fancy clients. We're in the thick of it, making things for the web.

Join over 3,000 web designers and web developers.

  • Advice and thoughts for people in the trenches.

Who am I?

My name is Sean and I'm probably a lot like you. I make stuff for the web. My web dev skills are mostly self-taught over the last decade or so. I really love what I do.

Today, I'm an entrepreneur. For the last five years I worked for an unecessarily secretive and mysterious industrial supply company. Before that I worked for a few years (and way too many hours) for a consulting company. I also have one of those fancy degree things in computer science.
